# 北京新能源车上牌流程

买车到上牌的整个流程：4S提车、办理发票、保险→办理临牌→发票工商验证盖章→购置税免税证明→网上或现场选号→检测场验车上牌。

# 1、4S提车/办理发票/上保险

  • 4S提车
  • 办理发票
  • 上保险：
    • 交强险（强制购买）：发生交通事故后，就能对车辆财产损失、人员伤亡等进行一个赔付；
    • 车损险：新手司机或昂贵车辆，推荐购买；
    • 第三者商业责任保险：发生事故造成他人人身伤害、财产损失时需要赔偿.可保多保少,很多人都是保100万；
    • 不计免赔特约险：购买者与公司商量协定好赔偿的条款，一旦出现交通事故的时候，当事人可以根据条款上的约定来进行赔偿；
    • 其他：自燃险、涉水险、划痕险、盗窃险

# 2、办理临牌

到4S店选定车辆后交尾款，然后在4S店内保险窗口上保险后，就可以拿着手中的所有材料去附近的交管局上临牌了，初次上临牌找离4S近的即可。

注：4S店基本上都会把后续需要的复印件都印好，自己也要留心检查资料列表有没有遗漏。

  • 首次办临牌需要准备的材料：
    • 身份证（暂住证或工作居住证）原件及复印件1份
    • 购车发票及复印件1份
    • 车辆合格证及复印件1份
    • 交强险凭证及复印件1份
  • 费用：10元
  • 上交：身份证复印件和交强险复印件；
  • 得到：临牌，有效期为7天，可以办3次，不受尾号限行政策。

注意： * 临时牌照要按照规定，一张粘贴在前挡风玻璃左下角或右下角。另一张粘贴在后挡风玻璃的左下角。 * 如不按照规定粘贴，将收到200元罚款并记12分的处罚。 >

# 3、工商验证

什么是工商验证：所谓工商验证又叫做工商备案，是售车单位所在的当地工商局对已售新车的录入备案，为以后此车进行做好准备。如所购新车在验车上牌前没有进行工商验证的话，并不影响随后的验车上牌的工作，但此车以后不能作为交易。原因就是当初没有进行工商验证，手续不全，不能予以过户。工商验证主要是为了车辆作为二手车出售。

拿到临牌后回到4s店就可以把车开去做工商验证了。

  • 需要的材料：车辆合格证、购车发票（第一联、第四联）原件、购车发票（第一联）复印件
  • 费用：6元
  • 上交：购车发票（第一联）复印件
  • 得到：车辆合格证、购车发票（第一联、第四联）原件上面均盖个工商已验的红章。

# 4、购置税免税证明

  • 北京市东城区国家税务局车辆购置税征收管理分局
  • 地址：东城区和平里6区18号楼前（十字路口西北角），咨询电话56091525
  • 办理免税证明需要的材料有：
    • 身份证及复印件
    • 车辆合格证及复印件
    • 发票第三联
    • 中签指标
    • 车辆购置税纳税申报表原件

# 5、网上或现场选号

参考：12123选号流程 (opens new window)

  • APP：交管12123
  • 准备工作：【业务中心】→【新车选号】→【车辆信息】→【预选号牌】
  • 选号方式：
    • 随机选号：可勾选不含数字"4"，有5次机会，每组产生10个号牌号码，任您选择。如果没有随机到心仪的号牌，您还可以选择自编选号方式。
    • 自选编号：总共可以自编50次，系统会按照录入顺序依次验证，注意把最心仪的号牌录在第一个。
  • 办理注册登记业务
    • 完成选号后，系统将显示选中号牌的有效期限，您需要在这之前办理注册登记业务。如果逾期未办理的，您选中的号牌将失效，且您将会被列入黑名单，且不允许在互联网上进行预选号牌号码，2年后将自动解除黑名单
