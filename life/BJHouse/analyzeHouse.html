# 首付90万北京房产1小时在线分析实录

此文是我个人帮朋友分析两套房子的过程，希望可以给有相关需要的朋友提供分析思路。

# 1、背景介绍

一位在豆瓣看过我买房经验文章的朋友，因为买房的事情，加了微信聊了比较多。

朋友在北京想买一套一居室，首付90万左右，选择非常有限。

看房一段时间后，有两套比较合适的房产，想让我帮忙分析一下，哪套更适合入手。

# 2、房产简介

# （1）尚东阁43平一居开间

一目了然，明厨暗卫，西向开间。

  • 明显缺点：厨房卫生间正对着，卫生间2平米，暗卫。

# （2）长城国际43平一居开间

一目了然，明厨明卫，东向开间。

  • 明显缺点：卫生间和厨房正对着。

# 3、多维度分析对比

首先，声明一下，这两套房子的户口都可以签走。

接下来，从住房各个维度进行详细的对比，如下表所示。

# 4、分析结果

通过以上表格对比来看：

  • 只有 目前租金和机场距离上，尚东阁 优秀；
  • 其他方面，如 周边环境（医院、商场、学校）、交通（高速口、地铁站、火车站）、户型、装修、未来发展潜力 长城国际都更加优秀。

最后，通过街景图对比来看，朝阳区双桥的尚东阁小区，受限于历史规划问题，整体还是村庄的气息。反观 通州区的 长城国际小区，小区环境、周围道路规划，以及未来通州作为城市副中心的地位，各方面都优于双桥。

所以，更加推荐 长城国际。

最终，朋友也参考我的建议，购买了通州的长城国际一居室，恭喜恭喜。

最后的最后，建议大家，买房是大事，其实分析起来并不太难，建议下定金签合同之前多做些功课，从实际入住之后未来几年的场景出发，踏踏实实分析，才有更多概率买到足够满意的房子。

备注：如果觉得文中有遗漏的购房考虑因素，大家可以留言提醒。
