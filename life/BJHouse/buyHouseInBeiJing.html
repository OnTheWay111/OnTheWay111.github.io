# 1、介绍

2012年，大学毕业，想法很丰满，想着从毕业开始一定要靠自己买房买车结婚。 2018年在我准备结婚的时候，没房没车没存款，房价从16年底暴涨至今。

其实，我只是个出身农村，父母都是农民的，一个最普通的90年打工人。

# 2、算账&筹钱

在北京买房，在老婆逼我算账之前，都是心怀恐惧，不敢奢望的。 但是，一算账发现，有这个可能性。

想计算自己首付的朋友，可以看第4部分

老婆愿意拿出全部积蓄+公积金+部分借款，我负责筹备剩下的部分。 开始行动，从亲戚朋友，到大学高中同学，到多年好友，到公司同事，能开口的都开口借了，1万不嫌少，10万不嫌多。

  • 亲戚：以年息3%借款，5年内还清。
  • 朋友：以年息6%借款，一年内还清。
  • 大学/高中同学：以年息5%借款，3年内还清。

所有借款均以个人名义借出，并说明个人及爱人收入情况，以及利息和还款期限。 诚实、诚恳、充满对新生活的希望，放下自尊，一个个打电话、上门见面，说明借还情况。 还好，可能大部分时间在朋友亲戚眼里，都还算靠谱。借钱成功率高达：70%。

账单如下：（虽然已尽力借钱，但是还是差了40万，最后不得已银行高息贷款） 

验证了那句话，成长从借钱开始

# 3、看房&买房

# 3.1 心理预期

买房最重要的是：明确心理预期（位置、总价、楼层、户型）+ 一个好中介；

# 3.2 多看多谈

其实刚开始，除了知道想买两居室和330万左右房子，其他什么预期都没有。 所以，整个看房从2月～6月，范围从北五环外（霍营、回龙观），南三环～南四环，东四环～东五环，前前后后换了6个链家中介，看了不下30套房子。 随着各种户型、楼层、位置、小区的看房，同中介聊天，找已买房前辈取经，加上自己多个条件筛选，基本上确定了房子画像。 

切记控制预算，坚持选房原则

房子画像如下：

  • 位置：东四环～东五环，近地铁
  • 房龄：95年之后
  • 户型及面积：60～70平，正规两居室
  • 楼层：2、3、4层
  • 配套：教育、休闲、医疗、绿化

# 3.3 找到合适中介

房子画像确定之后，相信你已经跟过不下于3、4个中介了，继续找，直到找到你信任、满意的人选。因为在我买房过程中，遇到过一心只为卖房型、老实但不够积极型、过于出谋划策型、怨天尤人型等等，一句话，对态度不满意的、不够积极的、不够信任的，果断换。直到找到各方面都达到满意程度的，最好是店铺销冠甚至区域销冠。我最终找的是一个92年的同龄人且是店铺和分区销冠，非常幸运。

说实话，链家应该是国内最靠谱的房产中介公司了，但是在链家买房被坑的事情还是层出不穷，绝大部分都是一个好中介可以帮你避坑的。（我找的销冠，因为比较年轻，有一套房子背景没有调查清楚，幸亏我花钱找了王牌经纪人帮我验房，才查出来户口及卖房的真实原因，不然也被坑了。）

# 3.4 找到目标房并谈价

找到合适的中介后，相信会有源源不断的合适房源推荐，记住你选房的原则，不要放弃，因为我觉得房子跟老婆一样，总会有属于你的最合适的那一个。

# 找到合适房源, 关注房源以下几点：

（1）户口必须全部可迁走；

（2）房子无抵押贷款（房贷以外）；

（3）房子未发生过火灾、凶杀；（火灾影响房屋质量、凶宅不吉利）

（4）邻居没有神经病；（小区里发现有神经病，每天上午乱骂人半天，管不了）

（5）房屋内不要有蟑螂；（蟑螂一旦有了永远消灭不干净）

（6）预算有限的情况下，目标房源肯定有明显的缺点，一定要找到他，并充分衡量利弊，是否可接受。交完定金后，就没有后悔药可吃了。

# 与房东谈价，注意：

（1）充分弄清楚房东卖房原因；（中间差点买到一套为还债卖房的，且户口无法迁走，买了就坑死了）

（2）看小区内一年内类似房源所有成交价格及成交均价；（对目标房源评估价格）

（3）对于我买的350万左右的房子，成交价一般比挂牌价低20万（砍价时尽量低于挂牌价20万开始谈）

（4）谈价期间，每次加价不要超过1万，谈钱不伤感情。（谈价格的过程中，其实也在考验双方的诚意，千万不要加价太快，讲究心理战。）

# 与中介谈中介费，注意：

对于总价较低房源，绿家中介一般收取2.7%的中介费。实在是太贵了，我们砍到了2.4%，大家加油砍，2020年成交量不多，应该可以更低。

# 3.5 付首付+拿房本

接下来就按照链家标准流程，一步步全力配合房屋交易，保障买房顺利。

# 4、买房公式

# 4.1 计算自己能凑到的钱

该部分就是首付

（1）自己有多少钱（包括公积金） （2）父母可以支持多少 （3）3年内除了房贷、生活费、保险、车等开支，你可以攒下多少钱？（这部分钱是你可以向亲朋好友借的钱，也是可以确保短期还得起的钱。） （4）如何是跟女朋友一起买，奔着结婚去的话，可以让你女朋友再按照上面的方法，算一下一共多少钱？

# 4.2 计算能买总价多少的房子

房子前提：满五唯一、无抵押、户口可迁走 虽然首套房首付30%，但是银行贷款评估价一般是房子市场价的90%～95%，再加上1%的税，2.5%左右的中介费，房子总价 ≈ 你能凑到的现金 / 45%。

# 5、总结

其实，越长大，越发现生活的不易，尤其是对于大部分普通家庭来说，在北京买房可能是父母及本人当前人生中挑战最大的一件事了。

所以，一定要多准备、多看、多聊、多谈，找个靠谱的中介，耐心看房，检查原则，希望大家都可以买到自己满意的房子。

生活总会以我们不曾认识的模样与我们相见，面对痛苦和迷茫，坚定的走下去吧。最后以伊隆马斯可的采访视频与大家共勉。(https://www.bilibili.com/video/BV1M54y1X72J)
