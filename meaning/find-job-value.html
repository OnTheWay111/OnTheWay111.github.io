# 一、前言

随着工作和生活经历的增长，难免会对工作感到迷茫。尤其是互联网行业，工作经验多了之后，不可避免的重复性工作和生活。为了更长远的规划生活和工作，找到职业核心价值观，是必要的。

# 二、我的职业价值观

我按照古典老师的《超级个体》课程中的方法，来找到自己的职业价值观。

以下为我的职业核心价值观测试过程，大家可以参考。

# 1、了解有哪些职业核心价值观

  • 利他主义：在于直接为大众的幸福和利益尽一份力。
  • 美感：能不断的追求美得东西，得到美感的享受。
  • 智力刺激：不断进行智力的操作，动脑思考，学习以及探索新事物，解决新问题。
  • 成就感：不断取得成就，不断得到领导和同事的赞扬，或者不断实现自己想要做的事。
  • 独立性：能够充分发挥自己的独立性和主动性，按照自己的方式，步调或者想法去做，不受他人的干扰。
  • 声望地位：所从事的工作在人们心中有较高的社会地位，从而使自己得到了人们的重视与尊敬。
  • 管理：获得对他人或某事物的管理支配权，能指挥和调遣一定范围内的人或事物。
  • 经济报酬：获得优厚的报酬，使自己有足够的财力去获得自己想要的东西，使生活过得较为富足。
  • 社会交往：能和各种人交往，建立比较广泛的社会联系和关系，甚至能和知名人物结实。
  • 舒适（环境）：希望能将工作作为一种消遣，休息或者享受的形式，追求比较舒适，轻松，自由，优越的工作条件和环境。
  • 安全感：不管自己能力怎样，希望在工作中有一个稳定的局面，不会因为奖金，涨工资，调动工作或领导训斥等经常提心吊胆，心烦意乱。
  • 人际交往：希望一起工作的大多数同事和领导人品较好，一起相处感到自然，愉快，认为这就是很有价值的事情，是一种极大的满足。
  • 追求新意：希望工作的内容经常变换，使工作和生活显得丰富多彩，不枯燥乏味。

# 2、选出最重要的8个

  • 选择8个：
    • 利他主义
    • 美感
    • 智力刺激
    • 成就感
    • 独立性
    • 管理
    • 经济报酬
    • 舒适（环境）
  • 不选择：
    • 声望地位：不期望得到人们的关注，更喜欢低调的幕后工作。
    • 社会交往：不喜欢和不认识的人交往，点头之交可以，多交流很尴尬。
    • 追求新意：不希望工作内容变来变去，导致无法深入成长，期望成为T字型人才。

# 3、删除3个

  • 删除3个：
    • 管理
    • 利他主义
    • 舒适（环境）
  • 剩余5个：
    • 美感
    • 智力刺激
    • 成就感
    • 独立性
    • 经济报酬

# 4、删除1个

  • 留下4个：
    • 美感
    • 成就感
    • 独立性
    • 经济报酬

# 5、删除1个

  • 留下3个，最后将这3个按照重要程度排序，这就是自己的核心职业价值观：
    • 成就感
    • 经济报酬
    • 独立性

# 6、尝试清晰的表述它们

举几个例子来说明你对这个词的理解

  • 成就感：做的事情受到别人的认可，做出的产品被别人喜欢。比如写的文章阅读、转发高，做的工具或平台使用度高。
  • 经济报酬：良好的经济回报，是保证持续投入的重要条件，且经济报酬也是重要的成就感之一。比如努力工作产出优秀，就要有对应的高回报。
  • 独立性：喜欢对事情的把控性，工作中喜欢独立空间创作和节奏把控。

# 7、评价当前工作的满意度

用这3个价值观评价当前的职业满意度。

  • 成就感：低，开发的工具和平台使用者较少，产品质量不够受重视；
  • 经济报酬：中等，工资中等水平；
  • 独立性：高，目前所做的事情，都可以自己完全设计和把控。

# 8、写下觉察和感悟

对我来说，工作中最重要的是成就感。没有成就感，很难有持续的热情投入到工作中。 对于我的工作属性，当有了成就感的时候，经济报酬很容易水涨船高。 足够的成就感表明产出有很高的认可度，那对工作的把控性也会比较高，前提是有一个愿意放权且懂行的leader。

通过以上职业核心价值观发现，当前互联网中的产品、开发、测试都属于比较适合的岗位。互联网有相对宽松的办公环境，良好的经济报酬，成就感也可以通过产出工具或平台来获得，尤其是面向C端的产品。

# 三、选择合适的职业

找到职业核心价值观之后，还需要选择合适的职业。

选择一个符合价值观的职业，不要为了财富或者收入而把自己"逼"进某一个职业里。

当我下午跟老婆漫步在小区的菜市场买菜时，越发觉得生活满意度和工作满意度是更加重要的。毕竟生活的最终目标是为了幸福，我们希望通过工作来打造幸福的人生，而不是工作本身。

有种观点，说选择一个你喜欢的职业。但是这个建议容易把人引入到，选择一个「羡慕」的职业上。所谓的合适，应该是能力、兴趣、乃至价值观的高度匹配。

(Stanley:) 我在《邻家的百万富翁》中说过，经济成功人士表现出一种不可思议的能力，虽然他们的教育或者经历并不顺利，但他们总能找到一份正确的工作。在这份工作中，他们收获了兴趣，也收获了经济回报。

不要把自己搞得压力太大，并不需要按照一条严谨的路线，高中、大学、毕业、工作才能找到属于自己的职业。多用演化思维，少想着「一个选择定终身」。该是你的，到头来还是会落到你的头上。
