# 1、命令行工具

# （1）Zellij-终端复用器 (opens new window)

一个终端复用器，类似于 Tmux，使用 Rust 开发的，功能更强大。

# （2）Linux 系统的磁盘分析工具 gdu 用法 (opens new window)

gdu 是 du 命令的 Go 语言版，用来分析 Linux 和 MacOS 系统的磁盘占用情况，提升了执行速度，并且有命令行图形界面。

# （3）远程终端控制工具: Termpair (opens new window)

一个远程终端控制工具。用户在一个终端窗口中，启动这个工具，它会返回一个链接，使用这个链接就能在其它计算机的浏览器里面，控制原始的终端窗口。

# （4）Mac桌面SSH客户端:nuoshell (opens new window)

一款 Mac 的桌面 SSH 客户端，集多功能于一身，简单、免费。

# （5）git用户管理器:gum (opens new window)

一个命令行的 Git 用户管理器，适合需要不同身份，向不同仓库提交代码的情况。

# 2、文本工具

# （1）MarkShow-markdown转PPT (opens new window)

一个在线工具，你输入 Markdown 文档，它自动转为 PPT 模式，进行在线演示。

# （2）Video Puppet (opens new window)

这个工具可以将一篇 Markdown 文档转成视频，支持插入图片和音频，还可以使用自动合成的语音解说。

# （3）Kakoune (opens new window)

一个类似 Vim 的文本编辑器，号称更易用、功能更强。

# 3、IDE工具

# （1）IntelliJ IDEA (opens new window)

Java开发最好的IDE工具，但是是付费的。

个人开发者可以选择破解版：

# 4、代码

# （1）use-sound (opens new window)

一个 React 钩子函数，用来添加声音效果，比如点击按钮发出“滴”的一声。

# （2）你不需要 GUI (opens new window)

这个仓库收集了一些常用的命令行操作，可以替代图形界面 GUI，主要针对初学者，但也适合作为命令行操作的备忘和速查。

# （3）web组件库: Shoelace (opens new window)

一个很不错的 Web Components 库，目前提供30多种网页组件。

# （4）识别人体动作: Handsfree.js (opens new window)

一个 JavaScript 库，可以识别人体的动作、手势和面部表情。官网上有很酷的演示，通过扭头或手势，控制鼠标移动。

# （5）USB激光传感器:Daytripper (opens new window)

这个开源项目是一个 USB 接口的激光传感器，能够感知周围物体的运动。作者举了一个很有趣的用例，老板走过来，它能立刻感知到，自动将计算机切换到工作桌面。

# 5、新技术

# （1）语音识别工具包Vosk (opens new window)

一个语音识别的工具包，支持18种语言（包括中文），体积较小（50MB），可以绑定各种语言，适合为聊天机器人提供语音识别。

# 6、信息

# （1）hacker-feeds-cli (opens new window)

一个命令行工具，可以查看 Hacker News、 Reddit、V2EX、GitHub Trending 等资讯网站。

# （2）Can I email (opens new window)

这个网站可以查询哪些前端 API 可以用于电子邮件。

# 7、图工具

# （1）开源工作流自动化绘图工具: n8n (opens new window)

一个开源的工作流自动化工具，可以通过图形界面，选择200多种服务，组成一个工作流。

# （2）图像无损放大工具:Ojoy (opens new window)

一个网页图像工具，可以放大图片的同时，还保持同样的清晰度。
